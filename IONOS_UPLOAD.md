# Subida a IONOS

Subir al directorio web de `negotecnica.net` solo estos elementos:

- `.htaccess`
- `index.html`
- `app.js`
- `styles.css`
- `data-store.js`
- `config.js`
- `register-sw.js`
- `sw.js`
- `manifest.webmanifest`
- `icons/`

No subir a IONOS:

- `.git/`
- `*.docx`
- `*.md`
- `config.example.js`

Despues de subir, abrir:

`https://negotecnica.net/?trackion-v=63`

Si el navegador venia usando una version anterior, cerrar y volver a abrir la app instalada.
